Animator

An Animator allows you to animate changes to the css properties of one or more DOM Elements. You can control the rate of playback using timing parameters, or create an interactive animation by scrubbing through the property values manually.

new Animator()

new constructor(target: (HTMLElement | NodeList), props: Object, timingParameters: TimingParameters?)
Parameters
target ((HTMLElement | NodeList)) The Element or NodeList being animated.
props (Object) The start and end values for the css properties being animated.
Name Description

timingParameters (TimingParameters? = {}) Playback and timing options.
Example
var animator = new Animator(box, {
     from: { translateX: 0,   rotate: 0   },
     to:   { translateX: 100, rotate: 360 }
}, {
     easing: 'spring',
     stiffness: 100,
     damping: 20
});
▸ progress

The value of this property (between 0.0 and 1.0) represents the progress of the Animator between its start and end states. The value is independent of time. For example, if an "ease-out" timing function is used, the value may increase more quickly at the start of an animation and slow down as it reaches its end. In some cases, the value may be less than 0.0 or greater than 1.0. When using spring physics, a property may overshoot its target value and oscillate before settling at an equilibirum position. Changing the value of this property interrupts the Animator and moves it to the "stopped" state.

progress
Example
console.log(animator.progress) // read
animator.progress = 0.5;       // write
▸ playState

The current play state of the Animator (Read Only). An Animator is "idle" when it is first created before any commands have been sent. After the "play" method is first called, it is in the "running" state when playing and in the "paused" state when paused. When an animation reaches its target value the Animator is in the "finished" state. If the stop method is called, the Animator is in the "stopped" state.

playState
Example
console.log(animator.playState); // e.g. 'running'
animator.playState = 'paused';   // NO EFFECT! - READ ONLY
▸ isReversed

If true, the Animator is animating in reverse, back towards its original values.

isReversed
Example
console.log(animator.isReversed); // e.g. false
animator.isReversed = true;       // play backwards

▸ continue(timingParameters = {})

Stop the animation and temporarily create a new one that proceeds in the same direction (based on the current value of "isReversed"). Useful if you need to interrupt an animation (For example during a user gesture), and send the target element towards the start or end state with a different set of timing parameters. When the new animation reaches its "finished" state, control is returned to the main timeline.

continue(timingParameters: TimingParameters?): Animator
Parameters
timingParameters (TimingParameters? = {}) Playback and timing options.
Returns
Animator:
Example

     easing: 'spring',
     stiffness: 100,
     damping: 20

TimingParameters

Timing parameters for an animation.

TimingParameters

Type: Object

Properties
easing (TimingFunction?) : The timing function to apply.
duration (Number?) : The duration for the animation. (Ignored if easing is set to 'spring').
stiffness (Number?) : The stiffness (spring constant) of the spring.
damping (Number?) : The damping (frictional force) applied to the spring.
velocity (Number?) : The initial velocity of the spring.
mass (Number?) : The mass of the spring.

TimingFunction

A timing function: "linear" | "ease" | "ease-in" | "ease-out" | "ease-in-out" | "spring"

TimingFunction

Type: String

AnimationEvent

An event fired by an Animator: "play" | "pause" | "finish"

AnimationEvent

Type: String

AnimationPropertyList

A set of css properties and their values, defining a state or key frame of an animation.

AnimationPropertyList

Type: Object

Properties
translateX (Length?) : translateX.
translateY (Length?) : translateY.
translateZ (Length?) : translateZ.
scale (Number?) : A shorthand for scaleX and scaleY. Ignored if another scale-related property is specified.
scaleX (Number?) : scaleX.
scaleY (Number?) : scaleY.
opacity (Number?) : opacity.
rotate (Angle?) : A shorthand for rotateZ. Ignored if another rotation-related property is specified.
rotateX (Angle?) : rotateX.
rotateY (Angle?) : rotateY.
rotateZ (Angle?) : rotateZ.
skew (Angle?) : A shorthand for skewX. Ignored if another skew-related property is specified.
skewX (Angle?) : skewX.
skewY (Angle?) : skewY.

Length

A Length is a data type available to certain animatable css properties. It can be specified as a number (of pixels) or a percentage. If no units are specified, px will be assumed.

Length

Type: (Number | String)

Example
...
{
     translateY: '50%',
     translateZ: 100
 }
...

Angle

An Angle is a data type available to certain animatable css properties. It can be specified as a number (of degrees), or in radians (e.g. '3.14rad').

Angle
